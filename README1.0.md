# 移动端Python代码编辑器

这是一个基于HTML、CSS和JavaScript的移动端Python代码编辑器，使用Pyodide在浏览器中运行Python代码。

## 功能特性

- **移动端适配**：专门为6.5-6.89英寸屏幕优化
- **Python代码编辑**：支持在移动端编辑Python代码
- **实时运行**：在浏览器中实时运行Python代码
- **版本检测**：自动检测GitHub上的最新版本
- **强制更新**：用户忽略3次更新后将强制更新
- **全面检测**：检查网络连接和本地存储

## 如何使用

1. 在浏览器中打开网站
2. 在代码编辑器中输入Python代码
3. 点击"运行"按钮执行代码
4. 查看运行结果
5. 当有新版本时，按照提示进行更新

## 技术栈

- HTML5
- CSS3
- JavaScript
- Pyodide (浏览器中的Python运行时)
